# Office 转 PDF 工具

一个简单易用的 Python 小程序，可批量将 Word、Excel、PPT 文件转换为 PDF 格式，支持 Microsoft Office 和 WPS 双版本。

## 功能特点

✅ **批量转换**：支持同时转换多个 Word/Excel/PPT 文件  
✅ **格式兼容**：支持 doc/docx、xls/xlsx、ppt/pptx 格式  
✅ **智能拆分**：Excel 文件按工作表自动拆分为独立 PDF  
✅ **双端支持**：兼容 Microsoft Office 和 WPS Office  
✅ **友好提示**：实时转换进度、成功/失败状态提示  
✅ **安全可靠**：完善的异常处理和资源释放机制  
✅ **中文支持**：完美处理中文文件名和路径  

## 环境要求

### 基础环境
- Python 3.6 及以上版本
- Windows 操作系统（依赖 Windows COM 组件）
- 已安装 Microsoft Office 或 WPS Office

### 依赖安装
```bash
# 安装必要依赖包
pip install pywin32
```

## 使用方法

### 1. 下载程序
将 `office2pdf.py` 文件保存到本地电脑

### 2. 运行程序
#### 方法一：命令行运行
```bash
# 打开命令提示符，进入程序所在目录
cd /d "程序所在文件夹路径"

# 运行程序
python office2pdf.py
```

#### 方法二：双击运行
直接双击 `office2pdf.py` 文件（需确保 Python 已添加到系统环境变量）

### 3. 操作步骤
1. 运行程序后，按提示输入待转换文件所在的文件夹路径（直接回车使用当前路径）
2. 程序自动扫描并统计可转换的 Office 文件数量
3. 确认是否开始转换（默认按 Y 确认）
4. 等待转换完成，PDF 文件会自动保存到原文件夹下的 `pdf` 子目录中
5. 转换完成后按回车键退出程序

## 目录结构
```
├── 待转换文件/          # 存放需要转换的Office文件
│   ├── 文档1.docx
│   ├── 表格1.xlsx
│   └── 演示1.pptx
├── office2pdf.py        # 主程序文件
└── pdf/                 # 转换后的PDF文件（自动生成）
    ├── 文档1.pdf
    ├── 表格1_工作表1.pdf
    ├── 表格1_工作表2.pdf
    └── 演示1.pdf
```

## 注意事项

⚠️ **重要提示**：
- 转换过程中请勿手动关闭 Office/WPS 进程
- 确保待转换文件未被其他程序打开或占用
- 转换前建议备份重要文件
- 空的 PPT 文件会被自动跳过
- 程序运行时需要 Office/WPS 正常安装并激活

❌ **常见问题**：
- 如果提示 "找不到路径"：检查输入的文件夹路径是否正确
- 如果转换失败：检查文件是否损坏或被占用
- 如果提示 COM 错误：确保已安装 Office/WPS 且正常运行
- 如果中文乱码：确保系统编码为 UTF-8

## 常见问题解答

### Q1: 程序运行闪退怎么办？
A1: 请通过命令行运行程序（`python office2pdf.py`），查看具体的错误提示信息

### Q2: Excel 转换后只有部分工作表？
A2: 程序会遍历所有工作表并生成对应的 PDF 文件，文件名格式为 `文件名_工作表X.pdf`

### Q3: 转换速度慢怎么办？
A3: 转换速度取决于文件大小和系统性能，建议分批转换大量文件

### Q4: 支持 Mac/Linux 系统吗？
A4: 暂不支持，该程序依赖 Windows COM 组件，仅适用于 Windows 系统
